# Juice Showcase App

This app is the repository-level showcase for Juice concepts.

Use it for:

- quickly browsing framework behaviors like selectors, relays, event subscriptions, and lifecycle cleanup
- validating widget-level integration patterns
- smoke-testing leased vs permanent showcase blocs

Do not treat it as the primary architecture reference. For production-style examples, start with:

- `packages/juice_examples/example/notes_app`
- `packages/juice_examples/example/social_feed`
- `packages/juice_examples/example/dashboard`
- `packages/juice_examples/example/ecommerce`
